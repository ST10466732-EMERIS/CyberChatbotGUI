# CyberGuard - Cybersecurity Awareness Chatbot

## Project Overview
This is a GUI Cybersecurity Awareness Chatbot built with WPF for Part 2.

## Features
- Modern WPF Interface
- ASCII Art on startup
- Voice greeting (greeting.wav)
- Keyword recognition (password, scam, phishing, privacy)
- Memory (remembers name and favourite topic)
- Sentiment Detection
- Random responses
- Error handling

## How to Run
1. Open solution in Visual Studio
2. Press F5
